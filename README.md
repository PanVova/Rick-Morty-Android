# Rick-Morty-Android


App for using Rick&Morty API
## Used Stack:
* Coroutines/Flow
* Conductor
* MVI
* Dagger2
* Clean Architecture
* Retrofit + Gson + OkHTTP
* Timber
* Epoxy
* Unit tests
* App shortcuts
* Caching - Room
